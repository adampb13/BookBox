# BookBox



## Aplikacja biblioteczna oparta na:

- Backend: Spring Boot (REST, JPA, Security)
- Frontend: React + Vite
- Baza danych: MySQL
- Orkiestracja: Docker Compose

## Struktura

- `backend/` – aplikacja Spring Boot
- `frontend/` – aplikacja React
- `docker-compose.yml` – uruchamianie całego stacku
